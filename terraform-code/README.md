# Terraform Code

This directory contains the Infrastructure as Code (IaC) used to deploy Azure resources for the Shadow Harbor Production Environment project.

## Files

### provider.tf
Configures the AzureRM provider and Terraform requirements.

### main.tf
Deploys an Azure Resource Group named:

rg-shadowharbor-tf-prod

located in East US.

## Skills Demonstrated

- Terraform
- Infrastructure as Code (IaC)
- Azure Resource Manager (ARM)
- Azure Resource Group Deployment
- Cloud Resource Provisioning
- State Management
